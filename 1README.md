# 8T Full Adder Design

## Project Description
This project implements a 1-bit Full Adder using optimized logic (8-transistor concept) in Verilog HDL.

## Inputs
- A
- B
- Cin (Carry In)

## Outputs
- Sum
- Cout (Carry Out)

## Features
- Simple full adder design
- Verilog HDL implementation
- Verified using testbench simulation

## Truth Table

0 1 0   | 1   0

- Any simulator (Vivado / ModelSim / Icarus Verilog)
